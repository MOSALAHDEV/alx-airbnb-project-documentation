# Airbnb Clone - User Stories

This document outlines the user stories for the Airbnb Clone, focusing on the needs of Guests and Hosts.

## Guest Stories
1.  **Registration:** "As a **Guest**, I want to **register an account using my email**, so that **I can access the platform to book homes.**"
2.  **Search:** "As a **Guest**, I want to **search for places by city and dates**, so that **I can find a property that fits my travel schedule.**"
3.  **Booking:** "As a **Guest**, I want to **book a property for specific dates**, so that **I can secure my accommodation.**"

## Host Stories
4.  **Create Listing:** "As a **Host**, I want to **create a new property listing with photos and price**, so that **guests can see what I am offering.**"
5.  **Manage Booking:** "As a **Host**, I want to **view and approve booking requests**, so that **I can control who stays at my property.**"

## Technical/Admin Stories
6.  **API Status:** "As a **Developer**, I want to **check the status of the API**, so that **I know the system is online and healthy.**"
